# Methodology

- Owner Earnings: conservative SBC expensing + maintenance capex parameterization.
- DCF: explicit forecast + terminal value (Gordon or exit multiple).
- Scenarios: Base/Bull/BlackSwan assumptions from YAML.
- Sensitivity: WACC x terminal g grid.
- Risk: Monte Carlo valuation-driver simulation and drawdown distribution.
- Portfolio: constrained Kelly with non-negativity and max-asset-cap.
