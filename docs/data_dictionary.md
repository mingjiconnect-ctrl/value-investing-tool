# Data Dictionary

## Units and conventions (validated in code)
- Currency: `USD` only for current implementation.
- Monetary unit: `million USD` in input fundamentals.csv.
- Time unit: `year` (annual frequency).
- Tax rate: decimal in [0,1].
- Shares outstanding: absolute share count.
- Net debt: monetary amount in same unit as enterprise value inputs.

## Key fields
- `revenue`: annual revenue.
- `net_income`: annual net income.
- `da`: depreciation & amortization.
- `capex`: total capex.
- `working_capital`: working capital stock level.
- `sbc`: stock-based compensation expense.
